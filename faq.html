Question-1: What are the different ways to select an element in the DOM?

Answer-1:We can select element Using ID, Class Name, Tag Name, Query Selector, Query Selector All, etc.

Question-2: What is the difference between innerHTML, innerText, and textContent?

Answer-2: The difference between innerHTML, textContent, and innerText lies in how they handle HTML tags, CSS visibility, and whitespace.

Question-3: What is event delegation in the DOM?

Answer-3: Event delegation is a technique in JavaScript where you attach a single event listener to a common parent element to manage events for all its child elements.

Question-4: What is event bubbling in the DOM?

Answer-4:Event bubbling is the DOM mechanism where an event triggered on a target element propagates upward through all of its ancestor elements to the document and window.

Question-5:How do you create, add, and remove element using JavaScript

Answer-5:To create a new element, use document.createElement(); to add it to the DOM, use a method like parent.appendChild(newElement); and to remove an existing element, use element.remove().
